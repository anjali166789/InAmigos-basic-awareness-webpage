# InAmigos-basic-awareness-webpage
## Overview

This project is a basic NGO awareness webpage created using HTML and CSS. The webpage aims to spread awareness about the initiatives, projects, and social impact of InAmigos Foundation.

## Features

* Responsive navigation bar
* Introduction to InAmigos Foundation
* Information about ongoing projects
* Social impact section
* Call-to-action section for volunteering and support
* Clean and user-friendly design
* Responsive layout for different screen sizes

## Technologies Used

* HTML5
* CSS3

## Sections Included

### About Us

Introduction to InAmigos Foundation and its mission.

### Ongoing Projects

* Project SEVA
* Project BACHPANSHALA
* Project JEEV
* Project UDAAN
* Project PRAKRITI
* Project VIKAS

### Social Impact

Highlights of the organization's contributions and achievements.

### Call to Action

Encourages visitors to volunteer, donate, and support the foundation's initiatives.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

Alternatively, use the Live Server extension in VS Code for a better development experience.

## Purpose

The purpose of this project is to create awareness about the work of InAmigos Foundation and encourage community participation in social welfare initiatives.
